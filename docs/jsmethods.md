## Arrays

### `push()`

Adds new element to the end of an array, returns length of modified array.

```javascript
let arr = ['red', 'blue', 'green'];
arr.push('black');
// 4
arr;
// ['red', 'blue', 'green', black]
```

### `pop()`

Removes last element of an array, returns the element that was removed.

```javascript
let arr = ['red', 'blue', 'green'];
arr.pop();
// 'green'
arr;
// ['red', 'blue']
```
